# 个股诊断模块 (Stock Analysis)

## 功能概述

个股诊断模块是一个专业的股票技术分析与投资建议系统，提供以下核心功能：

### 1. 主要指数展示
- 上证综指 (000001.SH)
- 深证成指 (399001.SZ)  
- 创业板指 (399006.SZ)
- 实时价格、涨跌幅显示

### 2. 股票搜索功能
- **中文关键词搜索**：使用股票名称字段进行模糊查询
- **股票代码搜索**：使用股票代码字段进行前缀匹配
- **拼音首字母搜索**：支持拼音转换后搜索
- **行业分类筛选**：按行业关键词搜索
- **股票全称精确匹配**

### 3. 股票详情分析
- **基本信息**：股票名称、代码、行业、地区、市场类型等
- **行情数据**：最新价格、涨跌幅、成交量、成交额等
- **K线图表**：日K、周K、月K线，支持缩放和移动
- **技术指标**：MACD、RSI、KDJ、BOLL等

### 4. 智能诊断系统
- **技术面评分**：基于价格趋势、技术指标的综合评分
- **基本面评分**：基础评分（可扩展更多财务数据）
- **综合评分**：技术面和基本面的加权评分
- **投资建议**：买入、持有、卖出建议
- **风险等级**：低风险、中等风险、高风险
- **详细分析**：具体的技术分析和风险提示

## 技术架构

### 后端技术栈
- **Django 4.2+**：Web框架
- **PyMySQL**：MySQL数据库连接
- **Pandas**：数据处理和分析
- **NumPy**：数值计算
- **PyPinyin**：中文拼音转换

### 前端技术栈
- **Vue 3**：前端框架
- **ECharts**：图表库，用于K线图和技术指标展示
- **响应式设计**：支持PC和移动端

### 数据库表结构
- `partition_table`：股票日线行情数据
- `股票基本信息remove`：股票基本信息
- `大盘指数`：主要指数数据
- `new_war_当前持股信息表`：持股信息
- `new_war_历史成交信息表`：交易记录
- `new_war_每日统计表`：每日统计

## API接口

### 1. 股票搜索
```
GET /api/stock_analysis/search/?q={搜索关键词}
```

**参数：**
- `q`: 搜索关键词（股票代码、名称、行业等）

**返回：**
```json
{
  "results": [
    {
      "st_code": "000001",
      "name": "平安银行",
      "industry": "银行",
      "area": "深圳",
      "market": "主板"
    }
  ]
}
```

### 2. 获取股票信息
```
GET /api/stock_analysis/stock/{股票代码}/
```

**返回：**
```json
{
  "basic_info": {
    "st_code": "000001",
    "name": "平安银行",
    "industry": "银行",
    "area": "深圳"
  },
  "daily_data": [...],
  "diagnosis": {
    "technical_score": 75,
    "overall_score": 70,
    "recommendation": "买入",
    "risk_level": "中等风险"
  }
}
```

### 3. 获取指数数据
```
GET /api/stock_analysis/index/
```

### 4. 获取图表数据
```
GET /api/stock_analysis/chart/{股票代码}/?period={周期}&days={天数}
```

**参数：**
- `period`: 图表周期（daily/weekly/monthly）
- `days`: 数据天数（默认120天）

## 技术指标说明

### MACD指标
- **金叉**：MACD线从下向上穿越信号线，买入信号
- **死叉**：MACD线从上向下穿越信号线，卖出信号

### RSI指标
- **超买**：RSI > 70，可能存在回调风险
- **超卖**：RSI < 30，可能存在反弹机会
- **正常**：30 ≤ RSI ≤ 70

### KDJ指标
- **金叉**：K线从下向上穿越D线，买入信号
- **死叉**：K线从上向下穿越D线，卖出信号

### BOLL指标
- **上轨**：股价触及布林带上轨，注意回调风险
- **下轨**：股价触及布林带下轨，可能存在反弹机会

## 注意事项

1. **数据更新**：确保数据库中的股票数据及时更新
2. **技术指标**：技术指标仅供参考，投资需谨慎
3. **风险提示**：股市有风险，投资需谨慎
4. **性能优化**：大量数据查询时注意数据库性能优化

## 故障排除

### 常见问题
1. **数据库连接失败**：检查数据库配置和网络连接
2. **图表不显示**：检查ECharts库是否正确加载
3. **搜索无结果**：检查数据库中的股票数据是否完整

### 日志查看
查看Django日志文件获取详细错误信息：
```bash
tail -f backend_main/backend/logs/debug.log
```
