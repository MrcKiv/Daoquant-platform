# 🚀 个股诊断系统快速启动指南

## 📋 系统要求

- Python 3.8+
- Node.js 16+
- MySQL 5.7+
- 网络连接（用于安装依赖包）

## 🎯 一键启动（推荐）

### 方法1：使用启动脚本
```bash
# 在项目根目录执行
python start_system.py
```

这个脚本会自动：
- 检查系统依赖
- 安装前后端依赖包
- 启动Django后端服务
- 启动Vue前端服务
- 打开浏览器访问个股诊断页面

## 🔧 手动启动

### 1. 安装后端依赖
```bash
cd backend_main
pip install -r requirements.txt
```

### 2. 启动后端服务
```bash
cd backend_main
python manage.py runserver 0.0.0.0:8000
```

### 3. 安装前端依赖
```bash
cd frontend
npm install
```

### 4. 启动前端服务
```bash
cd frontend
npm run dev
```

## 🌐 访问地址

- **前端页面**: http://localhost:5173
- **个股诊断**: http://localhost:5173/stock-diagnosis
- **后端API**: http://localhost:8000

## 📊 功能演示

### 1. 查看主要指数
页面顶部显示上证综指、深证成指、创业板指等主要指数的实时数据

### 2. 搜索股票
- 输入股票代码：`000001`
- 输入股票名称：`平安银行`
- 输入行业关键词：`银行`

### 3. 股票分析
选择股票后查看：
- 基本信息（行业、地区、市场类型）
- K线图表（支持日K、周K、月K）
- 技术指标（MACD、RSI、KDJ、BOLL）
- 智能诊断结果和投资建议

## ⚠️ 注意事项

1. **数据库配置**: 确保MySQL服务运行，连接配置正确
2. **端口占用**: 确保8000和5173端口未被占用
3. **网络环境**: 首次运行需要网络连接安装依赖包

## 🐛 常见问题

### Q: 后端启动失败
A: 检查MySQL服务是否运行，数据库连接配置是否正确

### Q: 前端启动失败
A: 检查Node.js版本，确保npm命令可用

### Q: 搜索无结果
A: 检查数据库中是否有股票数据，表结构是否正确

### Q: 图表不显示
A: 检查ECharts库是否正确加载，数据格式是否正确
