# 指数功能说明

## 🎯 功能概述

个股诊断模块现在支持在用户搜索股票前，默认显示主要指数的技术分析图表，用户可以点击切换查看不同的指数。

## 📊 主要指数

系统支持以下主要指数：

- **000001.SH** - 上证综指（默认显示）
- **000300.SH** - 沪深300
- **000905.SH** - 中证500
- **399001.SZ** - 深证成指
- **399006.SZ** - 创业板指
- **000688.SH** - 科创板50
- **000016.SH** - 上证50
- **000852.SH** - 中证1000

## 🔄 使用流程

### 1. 默认状态
- 页面加载时自动显示上证综指的技术分析图表
- 指数卡片显示最新价格和涨跌幅
- 当前选中的指数会高亮显示

### 2. 切换指数
- 点击任意指数卡片即可切换显示
- 切换后会自动加载该指数的技术分析图表
- 搜索框会清空，显示指数名称

### 3. 搜索股票
- 在搜索框输入股票代码、名称或行业关键词
- 选择股票后，指数图表会被股票图表替代
- 股票图表显示完整的技术分析指标

## 🎨 图表布局

### 指数图表（默认）
- **主图**: 指数走势 + 移动平均线(MA5/MA10/MA20)
- **副图1**: 成交量
- **副图2**: MACD指标
- **副图3**: RSI指标

### 股票图表（搜索后）
- **主图**: K线图 + 移动平均线 + 布林带
- **副图1**: 成交量
- **副图2**: MACD指标
- **副图3**: RSI指标

## 💻 技术实现

### 前端组件
- `stock-diagnosis.vue` - 主要的Vue组件
- 响应式数据管理
- ECharts图表渲染
- 指数切换逻辑

### 后端API
- `/api/stock_analysis/index/` - 获取指数列表
- `/api/stock_analysis/index/{ts_code}/` - 获取特定指数数据
- `/api/stock_analysis/search/` - 股票搜索
- `/api/stock_analysis/stock/{code}/` - 获取股票信息
- `/api/stock_analysis/chart/{code}/` - 获取股票图表数据

### 数据流程
1. 页面加载 → 获取指数列表 → 默认选中上证综指
2. 点击指数 → 加载指数数据 → 渲染指数图表
3. 搜索股票 → 加载股票数据 → 渲染股票图表
4. 返回指数 → 清除股票选择 → 显示指数图表

## 🔧 配置说明

### 指数配置
指数代码和名称映射在 `getIndexName` 函数中配置：

```javascript
const names = {
  '000001.SH': '上证综指',
  '000300.SH': '沪深300',
  // ... 更多指数
}
```

### 默认指数
在 `onMounted` 函数中设置默认选中的指数：

```javascript
// 默认选中上证综指，如果没有则选择第一个
const shanghaiIndex = indexData.value.find(index => index.ts_code === '000001.SH')
selectedIndex.value = shanghaiIndex ? shanghaiIndex.ts_code : indexData.value[0].ts_code
```

2. **性能优化**: 大量数据时考虑分页加载
